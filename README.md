# Personal Portfolio

A clean and responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills, projects, achievements, and contact links.

## Features

* Modern UI with mobile-friendly design
* Sections for projects, skills, and achievements
* Lightweight static website
* Deployed on Vercel

## Tech Stack

HTML • CSS • JavaScript

## Clone the Repository

```bash
git clone https://github.com/AdityaGade28/My-Portfolio
cd My-Portfolio
```

Open `index.html` in your browser.

## Live Demo

Hosted on Vercel.
